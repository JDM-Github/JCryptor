# JCryptor

JCryptor is used to Encrypt and Decrypt.

  Encrypting:

    cryp = JCryptor()
    original = "Hello World"
    >> Hello World
    encrypt = cryp.encrypt_text(original)
    >> G&uu) O)$u<
   
  Decrypting:

    decrypt = cryp.decrypt_text(encrypt)
    >> Hello World
